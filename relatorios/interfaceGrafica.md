# Ideia

Aplicar interfaces gráficas para melhorar interação com ambiente de testes.

### PySide6

Biblioteca em python que utiliza o framework Qt para criar interfaces gráficas. Permitindo estilizacao da interfaces com linguagem de estilo QSS, semelhante a CSS. Também possui bibliotecas especificas para fazer o processamento multi-thread e gerar gráficos em tempo real.
